  • HTML / CSS
    • HTML : 새로운 태그와 없어지는 태그, 버전별 업데이트와 권고사항을 공부합니다.
    • CSS : 하위버전에서 깨지는 CSS를 구분합니다.
    • CSS : calc, flex 등 새로운 CSS 공부 중하며 개인 연습용으로 활용하고 있습니다.
    • 블로그를 통해 인터넷에서 공부자료를 수집, 재구성해보며 공부하고 있습니다. 블로그 바로가기
  • JAVASCRIPT / JQUERY
    • 여러 특강과 강의를 다니며 기본 개념과 이해하고 혼자서 활용해 보는 중입니다.
    • 실제 업무상에선 메뉴, 아코디언, 탭, 배너 등 대체로 간단한 기능 정도만 직접 짜서 구현했었습니다.
    • 지역, 전역변수를 활용하고 Hoisting을 이해합니다.
    • ECMA SCRIPT 단계별 차이에 대한 이해가 부족하여 공부해 나갈 계획입니다.
  • 웹접근성, 웹표준
    • 웹 접근성 연구소의 "한국형 웹 콘텐츠 접근성 지침 2.1 (KWCAG 2.1)"를 읽었습니다.
    • K-WAH 프로그램, Color Contrast Analyser 사용하여 접근성 상태를 확인합니다.
    • 실무에선 사수에게 할 수 있는 일들을 나눠 받아 접근성을 맞추는 작업을 했었습니다.
    • W3C 웹표준 검사 사이트를 돌려 웹표준 준수를 확인하고 있습니다.
  • SCSS / GULP / GIT
    • 실무에서 한 두번 사용해본 것을 바탕으로, 혼자 공부하고 있습니다.
    • 전체적인 SCSS 문법을 이해하고 있습니다. SASS 기분 문법(TAB 들여쓰기로 맞추기)은 사용하지 않습니다.
    • 보통 WEB STORM IDE에서 GULP로 컴파일합니다.
    • 실무에서 소스 비공개가 무료로 가능한 bitbucket 사용 해보았습니다.
